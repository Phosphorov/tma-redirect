# Чат-бот для управления персоналом

## Описание проекта
Чат-бот для Telegram, интегрированный с Yandex Tracker, для управления персоналом, сменами, заявками и другими задачами. Проект использует систему ролей и хранит все данные в задачах Yandex Tracker.

## Архитектура
- Основной бот: `main_bot.py`
- Обработчики: `handlers/`
- Модели данных: `models/`
- Утилиты: `utils/`
- Конфигурация: `config/`

## Роли
- Администратор
- Руководитель
- Старший смены
- Сотрудник (штатный)
- Менеджер (аутстафф)
- Бригадир (аутстафф)
- Сотрудник (аутстафф)

## Установка
```bash
pip install -r requirements.txt
```

## Запуск
```bash
python main_bot.py
```
